# LocalAlert Pro

A React application for local alerts and facilities management.

## Deployment

This application is automatically deployed to GitHub Pages when changes are pushed to the main branch.

**Live Site:** https://debjitttdasss.github.io/campuss50forme/

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## GitHub Pages Setup

The deployment is handled automatically via GitHub Actions. To enable GitHub Pages:

1. Go to Repository Settings → Pages
2. Set Source to "GitHub Actions"
3. Push changes to the main branch to trigger deployment
